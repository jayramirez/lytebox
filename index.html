PARAMS

Object Type Default Description
GLOBAL PARAMS
params.top int auto position top of the popup content
params.width int 300 width popup content.
params.title string blank Heading title of the popup
params.message string copy, description, caption
params.align string left text alignment inside the popup. ('center','left','right')
params.closeButton boolean true some popups dont need a close button, set to false
params.escButton boolean true close popup with esc key
params.blurClose boolean true close popup by clicking the transparent background
params.addClass string blank add class attribute to popup content for custom style
params.fadeOut int [mili seconds] blank popup will auto close after the given time
params.onOpen function blank callback function after opening the popup
params.onClose function blank callback function after closing the popup
POPUP TYPES
params.type string ['alert', 'confirm'] alert = message box with 'ok' button,
confirm = message box with 'ok' and 'cancel' button
params.onConfirm function blank callback function when 'ok' button is clicked (requires params.type)
params.onCancel function blank callback function when 'cancel' button is clicked (requires params.type='confirm')
params.okCaption string 'Ok' caption of the ok button
params.cancelCaption string 'Cancel' caption of the caption button
params.buttonAlign string ['left', 'right', 'center'] right button alignment
EXTERNAL POPUP CONTENT
params.url string [url] url of page. (html,php)
params.data object blank data passed to the params.url. Fetch the data from the external page using $_GET

EXECUTION:

        popup.open(params);

MESSAGE POPUP

Simple message popup Demo

        popup.open({message: 'Hello world'})

added title parameter Demo

        popup.open({message: 'Hello world', title: 'Sample'})

added align parameter Demo

        popup.open({message: 'Hello world', title: 'Sample', align: 'center'})

short hand: this will only accept 1 string parameter Demo

        popup.open('hello world')

ALERT MESSAGE

added type parameter with 'alert' value: Demo

        popup.open({message: 'Hello world', title: 'Sample', type: 'alert'})

short hand: accepts 3 parameters, (message, string title [optional], callback function [optional] )
Demo 1 Demo 2 Demo 3 

        //demo 1
        popup.alert('hello world')

        //demo 2
        popup.alert('hello world', 'HEADING TITLE')

        //demo 3
        popup.alert('hello world', 'HEADING TITLE', function(){alert('ok button was clicked')})

CONFIRM MESSAGE

type parameter with 'confirm' value: Demo

        popup.open({message: 'Hello world', title: 'Sample', type: 'confirm'})

short hand: accepts 4 parameters
(message, string title [optional], on confirm callback function [optional], on cancel callback function [optional] )
Demo 1 Demo 2 Demo 3 

        //demo 1
        popup.confirm('hello world')

        //demo 2
        popup.confirm('hello world', 'HEADING TITLE')

        //demo 3
        popup.confirm('hello world', 'HEADING TITLE', function(){alert('confirmed')}, function(){alert('canceled')})

POPUP WITH EXTERNAL PAGE

Demo 1 Demo 2 
        //demo 1

        //demo 2
